# Texto curto para botão ou aba `Ajuda`

## Versão curta

Use o sistema nesta ordem:

1. Preencha `Professor Multiplicador`.
2. Cadastre as `Turmas`.
3. Registre os `Encontros`.
4. Revise `Checklist e relatório`.
5. Use `Acompanhamento de Cursistas` para busca ativa, ocorrências e socializações.

Lembre-se de:

- conferir a turma antes de salvar;
- evitar cadastro duplicado;
- fazer `Backup do banco` com frequência.

## Versão para painel interno

O `Multiplica Evidências` ajuda a organizar turmas, encontros, evidências, relatórios e acompanhamento de cursistas.

Para usar com segurança:

- escolha o mês correto;
- revise a turma antes de salvar;
- use filtros para localizar registros;
- faça backup com frequência.

Na aba `Acompanhamento de Cursistas`:

- `Cursistas`: cadastro e busca ativa;
- `Ocorrências`: problemas, encaminhamentos e pendências;
- `Socializações`: envio, apoio pedagógico, destaque, anexos e movimentações.

Em caso de dúvida, consulte o `Guia Rápido` ou o `Manual do Usuário`.
